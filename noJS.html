BONJOUR

 Votre navigateur ne peut pas interpréter le JavaScript qui est un langage employé dans ce site.

 Vous devez activer cette fonction pour pouvoir visiter ce site.

 Comment faire ? Dans votre navigateur, si c'est Internet Explorer, allez dans le menu "Outils", cliquez sur "Options Internet ...", cliquez sur l'onglet "Sécurité" puis sur le bouton "Personnaliser le  niveau". Avec l'ascenseur allez jusqu'à "Script - Active scripting" et cochez le bouton radio  "Activer". Faire "OK" partout.

Et ensuite ?
Les pages que vous consultez sont mémorisées dans le cache de votre navigateur. Il se peut, bien que vous ayez réactivé JavaScript, que vous ne puissiez toujours pas accéder aux pages en contenant. Vous devrez alors vider le cache présent sur votre disque dur.

- Sur Internet Explorer 5.5 et supérieur
Allez dans le menu "Outils", choisissez "Options Internet".
Selectionnez l'onglet "Général", puis la rubrique "Fichiers Internet temporaires" cliquez sur "Supprimer les fichiers".
Quittez votre navigateur et relancez Internet Explorer.


- Sur Netscape
Allez dans le menu "Edition / Préférence"
Selectionnez l'onglet "Avancées" puis "Cache"
Cliquez sur les options "Vider cache en mémoire" et "Vider cache sur le disque"
Quittez votre navigateur et relancez Netscape.

- Contrôle de la mise à jour des pages
Pour contrôler la mise à jour du site, assurez-vous que la mémoire "cache" de votre navigateur est correctement configurée (mise à jour des pages à chaque ouverture de page). Ce paramétrage permet de prendre en compte les mises à jour effectuées depuis votre dernière visite ou les modifications que vous effectuez sur le site en temps réel.
